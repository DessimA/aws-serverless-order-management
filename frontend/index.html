Termos de Uso

Projeto de Portfólio

CloudCert é um projeto de demonstração de arquitetura AWS serverless. Não realiza transações comerciais reais. Nenhum dado de pagamento é coletado ou processado.

Dados de Demonstração

As contas de usuário, pedidos e dados de catálogo são dados de demonstração. As informações podem ser redefinidas a qualquer momento sem aviso prévio.

Uso Permitido

Este sistema pode ser usado para avaliação técnica e testes de funcionalidade. Não use dados pessoais reais ao criar contas.
